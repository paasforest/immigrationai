## üìä KEYWORD DENSITY GUIDELINES

### Best Practices:
- **Primary Keyword**: Use 1-2% density (1-2 times per 100 words)
- **Secondary Keywords**: Use 0.5-1% density
- **Natural Integration**: Keywords should flow naturally
- **Avoid Keyword Stuffing**: Don't overuse keywords

### Where to Place Keywords:
1. **Page Title** (H1) - Include primary keyword
2. **Meta Description** - Include primary keyword
3. **First Paragraph** - Include primary keyword early
4. **Subheadings (H2, H3)** - Include related keywords
5. **Image Alt Text** - Include keywords
6. **URL Structure** - Include keywords (e.g., `/sop-generator`)
